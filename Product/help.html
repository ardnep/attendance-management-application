Attendance Management Application: Help Window

  • Creating A New Record: In order to create a new record, from the main menu go to the "New Record" window. There, by entering the name of the students you want included in your record and by specifying the name of the record, you can create a new record. After you have specified the name of students and record, click on the "Save" button. This will create a spreadsheet with ".xlsx" as its file format. You can access this newly created record through the "View Record" window accessible through the main menu.
  • Adding Attendance to A Record: To add attendance to a previously created record, simply go to the "Take Attendance" window using the main menu. There, you will be prompted to select the Zoom chat file you want to use to update your record. Students whose names have been mentioned in the chat file will be marked present. This chat file can be found in the Zoom folder inside the Documents folder after saving it using Zoom. After you select the chat file, you are required to enter the name of the record you want to update. Finally, click "Take Attendance" button to update your record.
  • Adding Names to A Record: To add names to a record, simply go to the "Update Records" window from the main menu. There, you can specify the names you want added along with the record in which you want to add these names. After that, you can click on the "Add Names" button to add the names you have mentioned in the record.
  • Removing Names from A Record: If you want to remove any names, similar to adding names, go to the "Update Records" window using the main menu. Then, add the names you want removed along with the name of the record you want removed. Finally, click on the "Remove Names" button to remove the names you want removed from the record.
  • Viewing Records: If you want to view any previously created records, then simply head to the "View Records" window from the main menu. There you will be able to search for records and will also see your recent searches. Additionally, you also have the option to view all the records that you have created.
